Django User Password Recovery / Reset Process
1️⃣ Access the Django Shell

Activate your virtual environment and navigate to your project folder (where manage.py is). Then run:

python manage.py shell


You’ll see the prompt:

(InteractiveConsole)
>>> 

2️⃣ List All Users

To see all usernames (and avoid typos or spacing issues), run:

from django.contrib.auth import get_user_model
User = get_user_model()

# List usernames
for u in User.objects.all():
    print(repr(u.username))  # repr shows exact spacing


Output example:

'admin'
'test'
'student'
'   anish'
'sofi'
'     sofiya'
'simran'
...


💡 Note: Extra spaces in usernames can cause “user does not exist” errors. Use the exact string or strip() it.

3️⃣ Reset a User’s Password

After identifying the correct username, reset the password in the shell:

# Replace with the exact username
u = User.objects.get(username='simran')  

# Set a new password
u.set_password('NewStrongPassword123')  

# Save changes
u.save()  

print('Password updated for', u.username)


✅ After this, the user can log in with the new password.
